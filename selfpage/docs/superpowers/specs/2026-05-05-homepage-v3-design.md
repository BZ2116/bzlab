# BZ's Lab v3.0 — 个人主页重设计

Date: 2026-05-05
Status: approved

## 概念与愿景

"晨雾"——一个如清晨薄雾般轻盈的个人主页。极淡的绿白渐变背景，细线时间轴贯穿页面，微妙的呼吸动效制造漂浮感。访客不是在"浏览网页"，而是在"穿过一个正在展开的故事"。整体传达：年轻、探索、清新、有潜力。

## 设计语言

### 色彩

| Token | Hex | 用途 |
|-------|-----|------|
| bg-top | #f8fdf9 | 页面顶部背景（暖调绿白） |
| bg-bottom | #f0f7f4 | 页面底部背景（冷调绿白） |
| timeline | #a5d6a7 | 时间轴细线 |
| node-default | #c8e6c9 | 节点默认态 |
| node-hover | #66bb6a | 节点 hover 态（更深绿） |
| glow | rgba(165, 214, 167, 0.4) | 节点光晕 |
| panel-bg | rgba(255, 255, 255, 0.85) | 底部 panel 磨砂白 |
| text-primary | #1a1a1a | 主要文字 |
| text-muted | #6b7280 | 次要文字 |

### 字体

- 标题/名字：无衬线粗体（如 Inter, system-ui fallback）
- 节点标签/副标题：等宽字体（如 JetBrains Mono, monospace fallback）
- 正文：同上无衬线，15px/16px

### 动效

- 节点呼吸脉冲：opacity 0.6 → 1.0，3s 循环 ease-in-out
- 节点 hover：scale 1 → 1.3，光晕扩散 200ms
- panel 滑入：translateY(20px) → translateY(0)，300ms ease-out
- panel 淡入：opacity 0 → 1，300ms ease-out（与滑入并行）
- 页面进入：整体 fadeIn 600ms

### 空间

- 页面最大宽度：960px，水平居中
- 时间轴位置：页面正中偏左（40% 处）
- panel 最大宽度：480px，水平居中，固定在视口下方 20%

## 交互行为

### Hover（轻触即现）

节点 hover 时：
1. 圆点 scale 1 → 1.3，出现光晕
2. 紧挨着节点出现一个小标签（2-3 个词），描述这个节点的核心信息
3. 标签在 hover 离开后 200ms 淡出消失

### 点击展开（NOTION block 风格）

1. 点击节点 → 底部 panel 从下方滑入
2. panel 内容：该节点的详细信息（根据节点不同内容不同）
3. 再次点击同一节点，或点击 panel 右上角 ×，panel 收起
4. 点击其他节点 → 当前 panel 收起，新 panel 滑入（无闪烁）

**节点 1（本科）Panel 内容：**
- 教育经历详情
- 主要项目：UAVP, MeDRNet
- 技术栈标签

**节点 2（硕士）Panel 内容：**
- 录取方向：网络安全
- 研究兴趣：低光视觉感知 / 目标检测 / 多模态分析 / CTF

**节点 3（Research）Panel 内容：**

## 响应式策略

- **桌面（>768px）**：时间轴居中偏左，节点标签左右交替
- **平板/手机（≤768px）**：时间轴整体左移，节点标签统一在右侧，panel 宽度占满屏幕

---

## 技术方案

- **框架**：沿用现有 Astro 项目，不更换
- **样式**：纯 CSS（不用 Tailwind），通过 CSS 变量管理颜色
- **动效**：CSS animation + 少量 vanilla JS 事件监听
- **Panel 展开**：纯 JS 控制 class 切换（.is-active）
- **无 React 组件**：首页用纯 Astro + JS，减少依赖复杂度
- **字体加载**：Google Fonts（Inter + JetBrains Mono），preconnect 优化
